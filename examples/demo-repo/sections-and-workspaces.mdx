# Sections and repositories

## Sections are folders

A section is a folder of notes. Create one from the `+` button in the sidebar (the **Section** tab), or hover a folder and use its `+` to add a note inside it. Folders collapse and expand in the tree.

This repository has two sections to look at:

- **projects/** — see [[projects/autonote-roadmap]]
- **daily/** — a date-based journal, e.g. [[daily/2026-06-19]]

You can nest a section inside another, and a note can live at the repository root or any depth.

## Repositories

The whole folder you opened is a **repository**. Use the switcher in the title bar to:

- **Open folder** — point Neuron at any existing folder of Markdown.
- **Create repository** — start a fresh one with a welcome note.
- **Recent** — jump back to a repository you used before.

<Callout type="info" title="Cloud is just a folder">
Put a repository inside OneDrive, Dropbox, or iCloud and it syncs like any other folder — Neuron marks it with a small cloud icon. No special setup.
</Callout>
