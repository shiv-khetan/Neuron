# MDX components

Beyond plain Markdown, Neuron ships live components you can drop straight into a note. They render in the preview pane.

## Badge

Badges take a `type` and `text`. The four types map to the app's semantic colors:

<Badge type="success" text="shipped" />
<Badge type="info" text="in review" />
<Badge type="warning" text="blocked" />
<Badge type="error" text="failing" />

## Callout

Callouts wrap a titled, colored block of text. Keep the body to a short sentence:

<Callout type="success" title="It saved">
Every keystroke is written to disk locally — there is no save button.
</Callout>
<Callout type="warning" title="Unclosed tags break the preview">
If you forget a closing tag, the preview shows a clear compilation error with a fix.
</Callout>
