# Demo repository

A ready-made Neuron repository that demonstrates every feature. It opens automatically the first time you launch the app.

## How to open it

1. Launch Neuron (`npm run dev`) — on first run this repository opens for you.
2. To reopen it later, use the **Repositories** page or the repository switcher in the title bar.
3. The folder lives at `examples/demo-repo`.

Then start at **getting-started** in the sidebar and follow the links.

## What's inside

| Note | Shows |
| --- | --- |
| getting-started | The hub — links, tags, a callout, a badge |
| markdown-basics | Headings, lists, quotes, code blocks, inline styles |
| mdx-components | The live `Counter`, `Badge`, and `Callout` components |
| wikilinks-and-tags | `[[wikilinks]]`, the graph, and `#tags` |
| sections-and-repositories | Folders, nesting, and repositories |
| plugins-and-ai | Enabling Claude, a local model, and Daily Notes |
| projects/autonote-roadmap | A note inside a section |
| daily/2026-06-19 | A dated journal entry in the `daily/` section |

## Try these

- Switch to the **Graph** tab to see the notes connected by their wikilinks.
- Click a `#tag` at the bottom of the sidebar to filter.
- Press **Ctrl/Cmd + K** to search notes and run commands.
- Open **Integrations & Plugins** at the bottom of the sidebar and enable a plugin — its panel appears on the right.
- Open the **Repositories** page (top of the sidebar) to manage every repository you've used.
