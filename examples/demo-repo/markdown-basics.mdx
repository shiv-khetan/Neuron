# Markdown basics

Neuron renders a focused subset of Markdown in the preview pane. Here is everything it understands.

## Headings

Use `#`, `##`, and `###` for the three heading levels. The note title is just an `#` heading at the top.

## Text styling

You can write **bold text**, *italic text*, and `inline code`. Mix them in a sentence and the preview keeps up as you type.

## Lists

- A bullet starts with a dash
- Each line is its own item
- Keep them short and scannable

## Quotes

> A blockquote is handy for callbacks, definitions, or a line you want to set apart.

## Code blocks

Fence code with triple backticks and an optional language:

```ts
function greet(name: string) {
  return `Hello, ${name}`;
}
```
