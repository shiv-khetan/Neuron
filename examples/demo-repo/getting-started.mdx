# Getting started

Welcome to Neuron. This workspace is a working tour of every feature — open the notes in order, edit them, and watch the preview on the right update live.

<Callout type="info" title="What is a workspace?">
A workspace is just this folder of Markdown files. It can live anywhere, including a OneDrive or Dropbox folder. Neuron never uploads your notes.
</Callout>

## The five-minute tour

1. Read [[markdown-basics]] to see how text renders.
2. Try the live components in [[mdx-components]].
3. Build a custom interface in [[building-htmx-views]].
4. Connect ideas with [[wikilinks-and-tags]].
5. Organize with [[sections-and-workspaces]].
6. Bring in AI and integrations from [[plugins-and-ai]].

## Where things live

- **Sidebar** — your notes and sections (folders). The `+` button creates a note or a section.
- **Title bar** — the workspace switcher, the `Search & commands` palette, the plugins button, and the panel toggles.
- **Right panel** — where enabled plugins show up (a Claude chat, a local model, a daily journal).

<Callout type="success" title="Tip">
Press Ctrl/Cmd + K anywhere to jump to a note or run a command.
</Callout>
